Project Gauss

Portable and Occlusion-Free Magnetic Object Tracking Using Analog Hall-Sensor Grid

Analog Hall-sensor grid, which functions as occlusion-free camera that only "see" magnetic objects,
is a new sensing technology that enables various novel features for interaction design.

Hardware Components

Each GaussSense board consists of 1 PCB board, 1 Microcontroller, 34 1-to-16 Multiplexers, and 16*32=512 Analog Hall Sensors.

  • PCB schemetic layout: (Filetype: Gerber, 4 layers) FREE LICENSE for Individual and Educational uses with non-commercial purposes.
  • Microcontroller: Teensy2, Arduino compatibles.
  • Multiplexer: NXP Semiconductors 74HCT4067PW (SOT355-1 Package).
  • Analog Hall Sensor: Winson Analog Hall Sensor: WSH136/WSH137/WSH138 (SOT-23 Package).

Firmware

  • Arduino Sketches: RawHID, UART.
  • Client: C++ (RawHID: XCode; UART: XCode), Processing (UART).
