GaussSense:
Portable and Occlusion-Free 3D Object Tracking Using Analog Hall-Sensor Grid

Since 2012, we have developed and presented several projects based on GaussSense, the analog Hall-sensor grid, which enables various exciting features for tangible and wearable interaction designs.
